# RMS REST APIs

##  School

- Get all schools   
```
GET - /schools
```

- Get school
```
GET - /schools/{schoolId}
```

- Create school
```
POST - /schools

Body:
{
  "name": "schoolName",
  "address": "address"
}
```

- Delete a school 
```
DELETE - /schools/{schoolId}
```
